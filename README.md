# Ozon Position Parser

Скрипт для поиска позиции товара (артикула SKU) в поисковой выдаче Ozon. Находит, на каком месте (1–100) находится товар, и возвращает результат в формате JSON.

## Требования

- Python 3.10+
- Playwright

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/caes4r/ozon-parser_list.git
cd ozon-parser_list
```
2. Создайте и активируйте виртуальное окружение:
```bash
python3 -m venv venv
source venv/bin/activate
```

3. Установите зависимости:
```bash
pip install playwright
playwright install chromium
```

## Использование

```bash
python parser.py "<поисковый запрос>" <артикул>
```

Пример:
```bash
python parser.py "нож туристический" 1871791848
```

## Вывод

Скрипт возвращает JSON-объект с полями:
- `query` – поисковый запрос
- `sku` – артикул
- `position` – номер позиции (1–100) или "not_found", если товар не найден, или "captcha" при обнаружении блокировки
- `page` – номер страницы, на которой найден товар
- `total_checked` – количество проверенных позиций
- `timestamp` – время выполнения в ISO формате

Пример успешного выполнения:

```json
{
  "query": "нож туристический",
  "sku": "1871791848",
  "position": 1,
  "page": 1,
  "total_checked": 1,
  "timestamp": "2026-03-24T15:22:10.081457"
}
```

## Тестовые данные

Для проверки использованы три товара по запросу «нож туристический»:

| Поисковый запрос  | Артикул (SKU) | Обоснование выбора                                                                                                                       |
|-------------------|---------------|------------------------------------------------------------------------------------------------------------------------------------------|
| нож туристический | 1871791848    | Мультитул тактический армейский – первый товар в выдаче (позиция 1). Проверяет корректность поиска на первой позиции.                    |
| нож туристический | 3515756446    | Нож охотничий для рыбалки с фиксированным лезвием (обычно позиция 3-5). Проверяет поиск товара на первой странице, но не в самом начале. |
| нож туристический | 2080598443    | Нож складной туристический (позиция 7+). Находится на второй странице, что проверяет работу пагинации.                                   |

Все три артикула гарантированно присутствуют в первых 100 позициях по запросу.

## Устойчивость

Для минимизации блокировок со стороны Ozon скрипт использует:

- Ротацию `User-Agent` из трёх распространённых строк.
- Расширенные HTTP-заголовки (`Accept-Language`, `Referer`, `DNT` и др.).
- Случайные задержки: 3–6 секунд после загрузки страницы, 5–10 секунд между страницами.
- Повторные попытки (до 2) при ошибках загрузки.
- Сохранение cookies между запусками для имитации «возвращающегося» пользователя.
- Обнаружение капчи по заголовку страницы (возвращается `"captcha"`).

## Ограничения

- Скрипт проверяет только первые 100 позиций (по условию задания).
- При частых запусках с одного IP может временно блокироваться Ozon; рекомендуется использовать прокси или менять IP.
- Структура страницы Ozon может измениться, что потребует корректировки селекторов.
